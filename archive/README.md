# Archived ImageJ Macros

This folder contains previous versions of the NMJ quantitative measures macros that are no longer actively maintained.

## Archived Files

- **NMJ_quantitative_measures_v2.6.ijm** - February 2019
  - Changed to use `lastIndexOf` for file extensions with multiple periods
  - Updated 3D Objects Counter plugin

- **NMJ_quantitative_measures_v2.6.1.ijm** - March 2019
  - Experimental threshold adjustments
  - Nearly identical to v2.6

- **NMJ_quantitative_measures_v2.7.ijm** - May 2019
  - Fixed rotation calculations in synaptic overlap
  - Removed 3D dispersion ratio calculation
  - Added en face area reporting
  - Changed output to CSV format
  - Fixed threshold at 10-255

## Current Version

Please use **NMJ_quantitative_measures_v3.0.ijm** in the parent directory, which includes:
- Manual threshold review capability (interactive adjustment)
- All features from v2.7
- Better usability and error handling

## Why These Are Archived

These versions have been superseded by v3.0 and are kept for:
- Historical reference
- Reproducibility of older analyses
- Comparison of algorithm changes

**Do not use these for new analyses unless you have a specific reason to replicate older results.**
